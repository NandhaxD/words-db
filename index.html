Word Guessing Game!

1. Read the Shuffled Word: Carefully read the shuffled word displayed on the screen. For example, "apple" might be shuffled to "alepp".

2. Assemble the Correct Word: Think carefully and try to unscramble the letters to form a correct English word.

3. Enter Your Answer: Type your answer in the input box provided.

4. Check Your Answer: If your answer is correct, you win! If not, you'll lose one life heart.

Lives: You start with 3 life hearts. Each time you enter an incorrect answer, you'll lose one life heart. If you lose all 3 life hearts, the game is over.

Game Over: If you lose all 3 life hearts, the game will end, and you'll need to start over.

Tips:
  • Take your time to read the shuffled word carefully.
  • Think logically and try to identify common letter patterns.
  • Don't be afraid to take a guess, but make sure you're confident in your answer!
